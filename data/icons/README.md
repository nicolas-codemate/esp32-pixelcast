# Icons Directory

Place your icon files here.

## Supported formats
- PNG (8x8 to 64x64 pixels, RGB/RGBA)
- GIF (static or animated)

## Naming convention
- Use lowercase
- No spaces (use underscore)
- No extension in API calls
