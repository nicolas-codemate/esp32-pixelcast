# GIFs Directory

Place your animated GIF files here.

## Constraints
- Maximum 64x64 pixels
- Maximum ~100 frames (memory dependent)
- Smaller is better for performance

## Optimization tips
1. Reduce frame count
2. Use fewer colors
3. Resize to exact display dimensions
4. Use [ezgif.com](https://ezgif.com/optimize) for optimization

## Naming convention
Same as icons - lowercase, no spaces.
